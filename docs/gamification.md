# Геймификационные механики для удержания клиента

## Цель
Повысить лояльность и частоту заходов в приложение с помощью игровых элементов, мотивирующих пользователя совершать целевые действия.

## 1. Уровни лояльности (прогресс-бар)

### Механика
Пользователь получает уровень в зависимости от суммы **рублёвого эквивалента** накопленного кэшбэка за всё время (формула из `loyalty-metrics.md`).

| Уровень | Порог (total_equivalent_rub) | Значок | Преимущества |
|---------|-------------------------------|--------|---------------|
| Бронза  | 0 – 5 000 ₽ | 🥉 | Базовый кэшбэк, участие в акциях |
| Серебро | 5 000 – 15 000 ₽ | 🥈 | +5% к кэшбэку у партнёров |
| Золото  | 15 000 – 30 000 ₽ | 🥇 | +10% к кэшбэку + приоритетная поддержка |
| Платина | > 30 000 ₽ | 💎 | +20% к кэшбэку + персональный менеджер |

### Визуализация в UI
- Крупный значок уровня с анимацией при переходе.
- Прогресс-бар: «До следующего уровня осталось 2350 ₽».
- Подсказка: «Совершайте покупки по картам и активируйте акции, чтобы быстрее расти».

### Реализация в API
Эндпоинт `GET /api/users/{user_id}/gamification` возвращает:
```json
{
  "level": "Бронза",
  "next_level": "Серебро",
  "progress": 0.42,
  "total_rub_cashback": 2100,
  "needed_for_next": 2900
}
