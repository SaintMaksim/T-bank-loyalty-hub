# AI-функционал раздела лояльности

## Цель

Повысить вовлечённость пользователя с помощью по-настоящему умных и персонализированных рекомендаций, генерируемых нейросетью на лету. Рекомендации помогают клиенту максимизировать выгоду от использования продуктов Т-Банка.

## Подход

Использованию генеративного ИИ через интеграцию с **Anthropic API**.

## Эндпоинт API

GET /ai/recommend

## Логика работы (псевдокод)

def get_ai_recommendation(user, current_offers): # 1. Собираем актуальные данные пользователя
segment = user.segment # LOW, MEDIUM, HIGH
balances = get_user_balances(user) # рубли, мили, браво
history = get_transactions_history(user, months=3)
forecast = calculate_forecast(history) # ожидаемый эквивалент в рублях

    # 2. Формируем строку с контекстом пользователя для промпта
    user_context = f"""
    Сегмент: {segment}
    Баланс: {balances.rub} руб, {balances.miles} миль, {balances.bravo} баллов Браво
    Прогноз на следующий месяц: {forecast} руб эквивалент
    Доступные офферы: {", ".join([offer.name + " " + offer.cashback + "%" for offer in current_offers])}
    """

    # 3. Настраиваем системный промпт
    system_prompt = (
        "Ты помощник по лояльности Т-Банка. "
        "Дай короткий персонализированный совет (2-3 предложения) "
        "как пользователю получить максимум выгоды."
    )

    # 4. Отправляем запрос в Anthropic API
    ai_response = anthropic_client.generate_text(
        system_instruction=system_prompt,
        prompt=user_context,
        temperature=0.7  # Немного креативности, но без лишних галлюцинаций
    )

    return ai_response.text

## Пример обработки запроса

### Входящий контекст (то, что улетает в API):

- **Сегмент:** HIGH
- **Баланс:** 3000 руб, 2000 миль, 500 баллов Браво
- **Прогноз на следующий месяц:** 1400 руб эквивалент
- **Доступные офферы:** МастерМинутка 10%, ЧистоБыстро 7%
