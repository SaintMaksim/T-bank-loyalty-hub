# AI-функционал раздела лояльности

## Цель
Повысить вовлечённость пользователя с помощью персонализированных рекомендаций, которые выглядят как «умные» советы от банка.

## Подход
Используется **rule-based система** (не машинное обучение), но в интерфейсе она подаётся как «AI помощник Т».  
Правила основаны на:
- финансовом сегменте пользователя (LOW / MEDIUM / HIGH)
- накопленных суммах по трём валютам (рубли, мили, баллы Браво)
- текущих акциях партнёров

## Эндпоинт API
`GET /api/users/{user_id}/ai-recommend`
