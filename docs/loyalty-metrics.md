# Метрики лояльности и прогноз

## 1. Совокупная лояльность (total_equivalent_rub)

Приводим все валюты к рублям:

total_equivalent_rub = rub_cashback + (bravo_points × 0.5) + (airline_miles × 2.0)

Где:
- rub_cashback – сумма начислений по программе Black
- bravo_points – сумма баллов Браво (1 балл ≈ 0.5 ₽)
- airline_miles – сумма миль (1 миля ≈ 2 ₽)

Пример:
Рубли=3000, Баллы=2000, Мили=1000
Итого = 3000 + (2000×0.5) + (1000×2) = 3000 + 1000 + 2000 = 6000 ₽

## 2. Прогноз на следующий месяц

Среднее за 3 месяца = (месяц1 + месяц2 + месяц3) / 3
Прогноз = Среднее × 1.2

Коэффициент 1.2 – предполагаемый рост активности после внедрения раздела лояльности.

Пример:
1500 + 1700 + 2000 = 5200; среднее = 1733; прогноз = 1733 × 1.2 ≈ 2080 ₽
